# ひらがな・カタカナマスター

## アプリケーション概要
「ひらがな・カタカナマスター」は、幼児や小学校低学年向けに作られた文字学習アプリケーションです。「マイ漢字タウン」のエンジンとアーキテクチャをベースに、まちづくり要素などの複雑な仕様を排除し、シンプルで直感的なUIに特化して再構築されています。

子供たちが楽しくひらがなやカタカナをなぞって学習し、習得した文字のかけらを使って新しい「ことば」を集めていくことで、自発的な学習を促すゲーミフィケーション・システムを採用しています。

## 主な機能

1. **文字学習 (なぞり書き・空書き)**
   - AnimCJKのデータを使用し、正しい書き順でのなぞり書きを判定します
   - 習熟度に応じて、ガイドあり（なぞり書き）からガイドなし（空書き）へとステップアップします

2. **ことばあつめパズル**
   - 学習して「自分のもの」になったひらがなブロックを組み合わせて、指定された言葉を探すパズルゲームです
   - 見つけた言葉は経験値になり、図鑑に記録されます

3. **ことばずかん**
   - 見つけ出した言葉をコレクションしていく機能です
   - ブラウザのSpeech Synthesis APIを用いた音声読み上げ機能を搭載し、耳からも学習をサポートします

## 開発・起動方法

### 前提条件
- Node.js (v18以上推奨)
- npm

### インストール＆起動
```bash
git clone [repository-url]
cd HIRAKATA_MASTER
npm install
npm run dev
```

ブラウザで `http://localhost:5173` を開くとプロトタイプが動作します。
